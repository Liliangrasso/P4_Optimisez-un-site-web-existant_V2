## Objectif du projet

Optimisé le Seo d'un site web 

## Pour commencer

### Contrainte technique

<ul>
  <li>Assurer l'accessibilité d'un site web</li>
  <li>Aucune erreur ni alerte au validateur W3C HTML et CSS</li>
  <li>Un affichage responsive</li>
</ul>
